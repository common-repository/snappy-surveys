== Description ==

	Get to know your audience. Create simple surveys that capture anonymous data. See insightful statistics from your surveys.

Some of the awesome features include:

*   Create unlimited surveys
*   Capture anonymous responses
*   See up to date stats and daily submission counts

== Installation ==

1. Unzip the plugin file
2. Upload the folder `snappy-surveys` and it's contents to the `/wp-content/plugins/` directory
3. Activate the plugin through the `Plugins` menu in WordPress
4. Use the `Snappy Survey` plugin menu to create a new  `Surveys`
5. Copy the Shortcode (ex. `[ssp_survey id="123"]`) next to your new survey and embed it anywhere in your website.
